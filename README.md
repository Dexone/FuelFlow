### Установка зависимостей

`npm install`

### Запуск проекта на локальном сервере

`npm run dev`

### Сборка проекта

`npm run build`

### Запуск собранного проекта

`npm start`

### Docker

`make up` (для запуска билда, вызов команды: `docker-compose up --build --remove-orphans`)
